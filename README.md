# Walmart Retail Analytics Dashboard

## Project Overview
This project is an interactive retail analytics dashboard built using MySQL, Power BI, DAX, and Excel. The dashboard analyzes Walmart sales data from 2010–2012 to identify sales trends, holiday impacts, store performance, and economic influences on retail sales.

---

## Objectives
- Analyze weekly and monthly sales trends
- Compare holiday vs non-holiday sales performance
- Identify top-performing Walmart stores
- Study economic indicators such as CPI, fuel price, and unemployment
- Build an interactive business intelligence dashboard

---

## Tools & Technologies Used
- MySQL
- Power BI
- DAX
- Excel

---

## Features
- Interactive KPI cards
- Weekly sales trend analysis
- Store-wise sales performance
- Holiday vs Non-Holiday comparison
- Economic indicator analysis
- Interactive slicers and filters
- SQL-integrated Power BI dashboard

---

## Dashboard Insights
- Holiday weeks showed higher average sales compared to non-holiday periods.
- Certain Walmart stores consistently outperformed others.
- Sales peaked during year-end seasonal periods.
- Economic indicators influenced retail performance trends.
